# Welcome to DataReachable Storybook

## 1. How to use this library?

This component library is developed based on react, so your need to install react
to using this component library.

```javascript
npm install react react-dom typescript@4.5 --save
```

your need to install this component library in your project and than use it. Please go to the @possie-engine/dr_pe_components_lib repo to see the detail usage steps.

```javascript
npm install @datareachable/dr_front_componentlibrary
```

You can check all the component api and how to import them in your project from Docs part.
